Software projects

These are selected applications and open-source projects I am building, maintaining or still consider relevant. Some solve concrete problems from my own work and infrastructure, some started as experiments, and all of them are available for others to use, inspect or build on.

Project Description Technologies
NASFiles A small, fast web file manager for home servers and small businesses. It keeps the filesystem as the source of truth while adding cloud-like previews, sharing, uploads, virtual SFTP, an S3-compatible API and secure local or OIDC authentication.
Vibe-coded around a real infrastructure need, then repeatedly security-audited and hardened.



ObsidiSync An Obsidian plugin for iOS, iPadOS and macOS plus a low-footprint Rust server that performs Git commits, rebases, pushes, conflict handling and file-history lookups without requiring Git on the device.
AI-assisted engineering with hands-on architecture, testing and operational hardening.


dbosk A native macOS database client for PostgreSQL, MySQL/MariaDB, MongoDB and SQLite. It supports streaming and cancellable queries, saved queries, CSV/JSON export, SSH tunnels and credential loading through shell scripts or AWS Secrets Manager.
AI-built with human-led product direction, architecture and validation.
Confluence Toolbelt Download Confluence pages as Markdown, edit them locally with Cursor, another AI agent or Obsidian, and upload or three-way-sync the result while preserving comments, mentions and collaborative changes. Available as an npm package and an Obsidian plugin.
Vibe-coded to solve a concrete documentation workflow.

ServoBill An open-source invoicing and expense platform for freelancers and small businesses. It supports PDF, ZUGFeRD and XRechnung invoices, AI-assisted expense import, reports, and both AWS serverless and Docker deployments.




PCBS Framework @tobisk/pcbs brings modern software-development practices to electronics engineering. Circuits are designed in TypeScript with type safety, modular composition and version control.
Mostly developed with AI.
