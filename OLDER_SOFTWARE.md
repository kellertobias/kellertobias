I regularly write software for my own projects, such as tools or automations   to simplify my daily life, or to help me or friends with work or hobbies.   Most of these projects never make it onto the main list because they are too   specific, too experimental or not mature enough to be useful for others.   Some examples include:

- Smart home automations in Node.js, for example a tool that fetches calendar     entries and sends them to my smart home system.

- A hardware controller for lighting and sound sources in an event venue.

- A LaTeX class for formatting the book Deadline     (https://www.tobisk.de/deadline) and my CV.

- This README generator, which builds my GitHub profile README from an HTML     source file.



## Rekordbox to Apple Music

[View repository](https://github.com/kellertobias/rekordbox-to-music-playlists)

A macOS tool for DJs that transferred Rekordbox playlists, ratings, BPM and   key information back into Apple Music. It used a custom Webpack setup to   compile TypeScript into the JavaScript dialect supported by AppleScript.

Status — completed legacy project, written in 2020 and finished in December 2021.

## Magic Cueboard

[View repository](https://github.com/kellertobias/magic-cueboard)

A companion interface for ChamSys MagicQ that extended the available   physical controls, communicated through OSC and the MagicQ HTTP API, and   displayed sound-pressure measurements from a GM1356 meter.

Status — superseded by newer show-control infrastructure.

## Calendar Sync

[View repository](https://github.com/kellertobias/calendar-sync)

## Past projects

- WordPress plugin for the TU Darmstadt theatre club to manage their     productions and team members on the theatre club website:     https://www.tud-schauspielstudio.de/technik/.

- Software written in Meteor with a custom connector to MySQL and LDAP for     user management of the same theatre club.

- Bachelor project: creation of worksheets for math classes, implemented as a     web-based application using LaTeX in the backend to generate PDF sheets.

- **[KUVO-Proxy](https://github.com/kellertobias/kuvo-proxy),     written for my own DJ streams in 2021**: got playing tracks out of     Rekordbox Performance mode in real time and translated the Rekordbox KUVO     calls to a real-time webpage for stream overlays. Rekordbox later dropped     KUVO support, so it no longer works.

- **[ToskLight](https://github.com/kellertobias/tosklight)**:     lighting control software for smaller clubs and theatres.

- **[ToskTVStudio](https://github.com/kellertobias/tosk-studio)**:     control suite for Blackmagic ATEM mixers and OBS for video streaming,     including tallies, text generators and the required hardware.

- **[ArtNET to OBS](https://github.com/kellertobias/artnet-to-obs)**:     used OBS as a media server for playing videos, images and music on a     screen, projector or stage, controlled by a lighting desk.
